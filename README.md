# 📌 Problem Statement

Small and medium manufacturers in India still manage operational sustainability data across:

- handwritten registers
- electricity bills
- WhatsApp invoices
- paper logs
- storekeeper notebooks

When buyers ask:

> “How much water was used per unit for this production batch?”

most factories cannot answer quickly, confidently, or with proof.

Trace solves this problem by converting scattered factory-floor data into:

# ✅ One honest, auditable sustainability score per production batch.

# 🏭 Core Features

## 1️⃣ Batch Setup
Create a new production batch in under 2 minutes.

### Fields:
- Batch ID
- Product Type
- Number of Units
- Start Date
- Primary Raw Material

Built specifically for low-friction factory-floor usage.

---

## 2️⃣ Resource Logging
Track operational resource consumption against each batch.

### Supported Resource Types

#### 💧 Water
- litres consumed
- source tracking
  - municipal
  - borewell
  - tanker

#### ⚡ Electricity
- kWh consumed
- solar contribution tracking

#### 🔥 Fuel / Thermal
- diesel
- LPG
- coal

#### ♻️ Waste
- solid waste quantity
- disposal method
  - landfill
  - recycler
  - composted
  - unknown

Supports:
- multiple entries per batch
- dated entries
- optional operational notes

---

## 3️⃣ Batch Report Card
When a batch is completed, Trace generates a printable report card showing:

### Included Metrics

✅ Total water consumption  
✅ Water per unit  
✅ Electricity consumption  
✅ Electricity per unit  
✅ Fuel consumption  
✅ Fuel per unit  
✅ Waste generated  
✅ Waste per unit  

### Operational Insights

Each report includes:

- comparison with previous batch
- deduction breakdown
- plain-language operational observation

Example:

> “Water consumption per unit increased 18% compared to your previous garment batch.”

---

## 4️⃣ Honest Sustainability Score

At the top of every report card:

# 🎯 Sustainability Score / 100

The scoring formula is implemented EXACTLY as specified in the challenge.

---

## 📐 Scoring Formula

Start at **100**.

### Deductions

| Condition | Deduction |
|---|---|
| Water per unit above 50L | 1 point per 10L |
| Electricity per unit above 2kWh | 1 point per 0.5kWh |
| Any fuel usage | -10 |
| Landfill / unknown waste | -15 |
| Recycler waste | -5 |
| Tanker water used | -10 |

### Final Score

- Minimum: 0
- Maximum: 100

No gamification.
No badges.
No AI-generated scoring.

Just transparent operational math.
